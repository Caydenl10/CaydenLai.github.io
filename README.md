# Cayden Lai — ECE ePortfolio

Personal portfolio website for **Cayden Lai**, B.S. Computer Engineering student at Georgia Institute of Technology.

**Live Site:** https://caydenl10.github.io/CaydenLai.github.io/

---

## About

This is my personal ePortfolio, built from scratch using plain HTML, CSS, and vanilla JavaScript — no frameworks, no build tools, no dependencies. It serves as a professional showcase of my engineering projects, academic journey, and career goals.
