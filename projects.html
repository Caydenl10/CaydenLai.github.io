Individual Discovery Project Spring 2026

Arduino Temperature-Responsive Cooling System

I built an Arduino-based cooling system that uses a TMP36 analog temperature sensor to measure ambient temperature in real time and automatically adjusts fan speed using PWM motor control. The system uses a logic-level N-channel MOSFET for safe power switching and a flyback diode for motor protection. I implemented the embedded firmware to read sensor data through the Arduino's ADC, convert the voltage output to a temperature value, and map the measured temperature to a variable PWM duty cycle — creating a closed-loop system that responds dynamically to its environment. This project gave me direct, hands-on experience with the core ECE fundamentals I'm studying at Georgia Tech: analog signal acquisition, digital-to-analog interfacing, and embedded systems programming — and reinforced how hardware and software must be co-designed to build reliable real-world systems.

Built with
Arduino C/C++ PWM TMP36 MOSFET

Hackathon Project February 2026

bacter.ai — Antibiotic Resistance Prediction

bacter.ai is a machine learning web application that predicts resistance or susceptibility across 10 major antibiotics from a raw bacterial DNA sequence in seconds, giving doctors actionable treatment guidance before traditional 48–72 hour lab results are available — critical for sepsis patients where every hour on the wrong antibiotic increases mortality by 4–8%. Built in 36 hours, we trained 10 XGBoost classifiers on 3,030 lab-verified E. coli genomes using k-mer frequency extraction, achieving up to 85% accuracy with 94.6% sensitivity, with SHAP-based explainability highlighting the exact DNA patterns driving each prediction. This project reinforced core ECE concepts around signal processing and feature engineering, and showed me firsthand how machine learning pipelines — from data acquisition and model training to real-time API deployment — apply directly to biomedical computing problems I want to pursue in my career.

Built with
Python XGBoost scikit-learn SHAP Flask React D3.js pandas

Individual Project May 2025

Runner Management System

The Runner Management System is a JavaFX desktop application connected to a live MySQL database, designed for coaches or athletic organizations to manage a full roster of runners. It supports complete CRUD operations — insert, update, delete, and browse records — along with live search and filter by name or gender, multi-step record navigation, and one-click export of the entire dataset to a formatted Excel spreadsheet. Building this project gave me hands-on experience connecting an object-oriented Java application to a relational database via JDBC, designing event-driven GUIs using FXML and the MVC pattern, and writing a clean data access layer — core software engineering skills that directly apply to embedded systems, data pipelines, and full-stack ECE applications.

Built with
Java JavaFX MySQL JDBC Apache POI Maven
